# Cloudbox API  

Node/Express server that queries MongoDB Atlas and holds files for the Cloudbox App. An Auth0 JWT is required to access all routes.

## Routes  

/files  
`Post`  
`Patch`  

/files/:email  
`Get`  

/files/:id  
`Delete`  

/files/folder  
`Post`  

/user  
`Get`  
`Put`  

/user/groups  
`Patch`  

/user/:email  
`Get`
